# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in CursorMeter, **please do not open a public issue.** Instead, report it privately:

**Email**: woojin.skp@gmail.com

### What to include

- Description of the vulnerability
- Steps to reproduce
- Expected impact

### Process

1. You report privately via email
2. I acknowledge receipt within 48 hours
3. Fix is developed and tested
4. New release is published
5. Vulnerability is disclosed publicly

Thank you for helping keep CursorMeter secure.
